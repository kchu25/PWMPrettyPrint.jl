PWMPrettyPrint.logoshow — Function
logoshow([io,] pfm; background, rna, height, col_width)

Render a sequence logo directly in the terminal using braille sub-pixel characters for high resolution and uniform solid colours (no anti-aliasing blending).

Each braille character encodes a 2 × 4 pixel block, giving effectively twice the horizontal and four times the vertical pixel density of ordinary text, while remaining fast (binary rasterisation, O(pixels) with no supersampling).

Arguments

  • pfm – 4 × L position-frequency matrix (rows = A/C/G/T).
  • background– background frequencies (default: uniform 0.25).
  • rna – use A/C/G/U alphabet (default false).
  • height – terminal rows (default 20; pixel height = 4 × height).
  • col_width – pixel columns per position, must be even (default 12).
